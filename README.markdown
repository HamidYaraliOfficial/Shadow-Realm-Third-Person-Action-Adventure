# Shadow Realm – Third-Person Action Adventure

A modern desktop action-adventure game developed with Python and PyQt6. Players battle waves of enemies, explore multiple worlds, level up their hero, unlock stronger abilities, and survive increasingly difficult encounters.

## Features

- Third-person action gameplay
- Multiple worlds (Forest, Dungeon, Castle)
- Hero progression and leveling system
- Enemy waves and boss battles
- Skills including Attack, Dash, and Spell
- Particle effects and visual animations
- Dark and Light themes
- Multi-language support (English, Persian, Chinese)
- Dynamic HUD and score system
- Keyboard and mouse controls

## Requirements

- Python 3.10 or newer
- PyQt6

## Install Required Libraries

```bash
pip install PyQt6
```

## Run

```bash
python "Third-Person Action Adventure.py"
```

## Controls

| Key | Action |
|------|---------|
| W A S D | Move |
| Space | Attack |
| Shift | Dash |
| E | Spell |
| Left Mouse | Attack |
| Right Mouse | Spell |
| Esc / P | Pause |

## Gameplay

1. Start a new game.
2. Select a world.
3. Defeat enemies and survive waves.
4. Gain experience and level up.
5. Defeat bosses.
6. Complete all waves to achieve victory.

# 🇨🇳 中文

# 暗影王国（Shadow Realm）- 第三人称动作冒险游戏

这是一个使用 Python 和 PyQt6 开发的桌面动作冒险游戏。玩家需要在多个世界中探索、击败敌人、挑战 Boss，并不断提升角色能力。

## 功能特点

- 第三人称动作玩法
- 多种地图世界（森林、地牢、城堡）
- 角色升级系统
- 敌人波次与 Boss 战斗
- 攻击、冲刺与魔法技能
- 粒子特效与动画系统
- 深色与浅色主题
- 多语言支持
- 实时 HUD 界面
- 键盘与鼠标控制

## 系统要求

- Python 3.10 或更高版本
- PyQt6

## 安装依赖

```bash
pip install PyQt6
```

## 运行程序

```bash
python "Third-Person Action Adventure.py"
```

## 操作说明

| 按键 | 功能 |
|------|------|
| W A S D | 移动 |
| Space | 攻击 |
| Shift | 冲刺 |
| E | 魔法 |
| 鼠标左键 | 攻击 |
| 鼠标右键 | 魔法 |
| Esc / P | 暂停 |

## 游戏流程

1. 开始新游戏。
2. 选择游戏世界。
3. 击败敌人并完成波次挑战。
4. 获取经验并升级。
5. 击败 Boss。
6. 完成全部关卡获得胜利。

## 使用技术

- Python
- PyQt6
- 面向对象编程
- 自定义渲染系统
- 实时游戏循环
